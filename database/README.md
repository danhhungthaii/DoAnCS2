# Database Schemas & Models

## Collections

### 1. Users (Admins)
- Quản lý tài khoản admin đăng nhập

### 2. Events
- Thông tin sự kiện
- Tọa độ GPS (latitude, longitude)
- Bán kính check-in cho phép

### 3. Students
- Danh sách sinh viên
- Mã sinh viên, tên, lớp, email

### 4. Attendances
- Bản ghi điểm danh
- Liên kết Event - Student
- Thời gian check-in
- Tọa độ GPS khi check-in
